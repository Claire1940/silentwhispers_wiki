<Section icon={Sparkles} title="Unreal Engine 5によるゲームプレイの特徴">

『コードネーム：神不言』の配信が他の恋愛ゲームと一線を画しているのは、Unreal Engine 5（UE5）を全面的に採用している点です。VVANNA STUDIOが公開した「実機プレイ映像」では、4Kテクスチャ、ダイナミックライティング、緻密な衣服の物理演算など、このジャンルでは類を見ないレベルのディテールが披露されました。

ゲームプレイは主に「都市探索」と「ストーリー進行」の2つのサイクルに分かれています。

**ゲームプレイの核となる要素:**
1. **自由度の高い都市探索**: カフェ、空中駅、観覧車など、「蝕光都市」の中を自由に巡ることができます。
2. **シネマティックな交流**: リアルタイムでレンダリングされる、高精度な表情アニメーションを交えた親密なシーン。
3. **タイムループストーリー**: 「真実」の道を見つけ出すために、特定の出来事を繰り返さなければならない複雑な物語構造。
4. **生存を懸けた選択**: 生存、裏切り、そして感情を揺さぶる結果を伴う、極めて重要な決断。

| 特徴 | 技術的な実装 | プレイヤーへの影響 |
| :--- | :--- | :--- |
| **ビジュアル** | **UE5 4Kレンダリング** | 圧倒的な没入感とリアリズム |
| **探索** | **オープンな都市ハブ** | 非線形な発見の楽しさ |
| **ストーリー** | **分岐するループ** | 高いリプレイ性（周回プレイ価値） |
| **オーディオ** | **立体音響 / OST** | 感情を揺さぶる雰囲気作り |

<Callout type="info" title="動作環境について">
ゲームはモバイル向けに最適化されていますが、実機デモで披露されたフル4Kのシネマティック効果を体験するには、最新のGPUを搭載したPCが必要となります。
</Callout>

</Section>

<Section icon={Sparkles} title="Unreal Engine 5のゲームプレイと特徴">

**コードネーム：神不言**の技術的な実力は、最も注目されている特徴の一つです。**Unreal Engine 5**を活用することで、肌の質感、布地の物理演算、環境ライティングにおいて、乙女ゲーム市場では稀に見るレベルのディテールを実現しています。この「4Kシネマティック」アプローチにより、ストーリーカットシーンとインタラクティブなゲームプレイの間でシームレスな移行が可能になっています。

| 特徴 | カテゴリー | 説明 |
| :--- | :--- | :--- |
| **4Kビジュアル** | **グラフィック** | キャラクターや街並みの高精度な3Dレンダリング。 |
| **一人称インタラクション** | **没入感** | キャラクターと直接触れ合う親密なクローズアップシーン。 |
| **都市探索** | **ワールド** | カフェ、空中駅、公園などを自由に移動可能。 |
| **タイムループ叙事詩** | **ストーリー** | 生存のためにタイムラインをリセットまたは変更する選択。 |
| **危険なゲーム** | **対立** | 裏切りや決断が絡む、生死を賭けたシナリオ。 |

ゲームプレイは、**「日常のロマンス」**と**「生存の試練」**という2つのメインループに分かれています。日中、プレイヤーは「光の消えゆく街」を探索し、デートを楽しみ、関係を築くことができます。しかし、「危険なゲーム」フェーズではサバイバルメカニクスが導入され、プレイヤーは自分自身と選んだパートナーを守るために、幻想と現実の間を切り抜けなければなりません。

</Callout>

<Section icon={Shield} title="ストーリー設定：光の消えゆく街">

**コードネーム：神不言**の舞台は単なる背景ではなく、ゲームのルールを支配する生きた存在です。**「光の消えゆく街」**（City of Fading Light）は、現実と幻想の境界が曖昧なファンタジーメトロポリスです。プレイヤーは「繰り返される生」のサイクルに囚われ、死や失敗のたびにループが再開され、過ちを正したり隠された裏切りを暴いたりする二度目のチャンスが与えられます。

**主要な世界観の要素：**
- **境界（The Boundary）**: 通常の都市と、危険な影の「迷宮」を隔てる薄いヴェール。
- **生存ゲーム（The Survival Games）**: 住民が自らの存在を賭けて競い合うことを強いられる定期的な試練。
- **タイムループ**: プレイヤーキャラクターが異なる現実のバージョンを越えて記憶を保持できる物語上の仕掛け。
- **都市探索**: 空中駅や夕暮れの観覧車などの場所は、ロマンチックなデートの場であると同時に、物語の重要な転換点にもなります。

| 場所 | カテゴリー | 重要性 |
| :--- | :--- | :--- |
| **空中駅** | **移動** | 都市探索やキャラクターとの出会いの拠点。 |
| **観覧車** | **ロマンス** | シネマティックなデートや夕日の景色を楽しめる象徴的なスポット。 |
| **市街地のカフェ** | **日常生活** | カジュアルな交流や噂話を集めるためのエリア。 |
| **影の迷宮** | **危険** | 生存ゲームが行われる、生死を賭けたエリア。 |

<Callout type="tip" title="世界観の考察">
ワールドプレミアPVのセリフに注目してください。「本物の愛まで、あと一筋の囁き」というフレーズは、世界の幻想的な性質と、ループの中で真の繋がりを見つけることの難しさを暗示しています。
</Callout>

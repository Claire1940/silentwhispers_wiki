<Section icon={Shield} title="ストーリー設定：没光の街">

**コードネーム：神不言**の物語は、現実と幻影の境界が曖昧になり始めた世界で展開されます。没光の街は、表面上は普通の賑やかな大都市に見えますが、プレイヤーが生き残るためにマスターしなければならない「奇妙なルール」の下で機能しています。日常（カフェでコーヒーを飲むこと）と非日常（生死をかけたゲームへの参加）のコントラストが、プロットの中心的な緊張感を生み出しています。

公式トレーラーで言及されている「危険なゲーム」は、この恋愛が単なる甘い瞬間だけでなく、共有されたトラウマや生存をテーマにしていることを示唆しています。3人の男性主人公はそれぞれ、これらのゲームと異なる関係を持っています。ある者はそれを執行し、ある者はそのサイクルの犠牲者となっています。

| 場所 | カテゴリ | 重要性 |
| :--- | :--- | :--- |
| **空中ステーション** | 交通 | 都市全体の探索拠点 |
| **観覧車** | ランドマーク | 重要な恋愛シーンの舞台 |
| **街のカフェ** | 交流 | 日常的な交流のメインスポット |
| **影の迷宮** | 謎 | タイムループ現象の源 |

<Callout type="tip" title="物語の戦略">
没光の街の背景の細部に注意を払ってください。照明や時間帯は、そこが「安全地帯」なのか、あるいは「サバイバルの試練」が近づいているのかを示す合図となることがよくあります。
</Callout>

</Section>

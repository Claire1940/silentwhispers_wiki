よくある誤解として、事前登録時に受け取るメール認証コードが挙げられます [2]。このコードはメールアドレスの有効性を確認し、アカウントを保護するためのものであり、ゲーム内アイテムと交換できるギフトコードではありません [2]。無関係なゲームのコードを使い回しているような、非公式サイトの「有効コード一覧」などには十分ご注意ください [1, 2]。

| 特徴 | ウェブサイト認証コード | 公式プロモーション/ギフトコード |
| :--- | :--- | :--- |
| 目的 | アカウントのセキュリティと設定 | 無料のゲーム内アイテムや通貨の獲得 |
| 配布元 | 公式サイトの登録フォーム | 公式SNSなどのキャンペーン |
| 送付場所 | 登録したメールアドレスに直接送信 | 公式X（旧Twitter）やDiscordで一般公開 |
| 使用方法 | 登録時に入力 | 正式リリース後にゲーム内で入力 |
| 現在の状況 | アカウント作成時に有効 | 正式リリース後に配信予定 |

<Section icon={Map} title="システム要件とUE5の最適化">

**コードネーム：神不言 Mac版**の体験は、**Unreal Engine 5**の採用に大きく左右されます。このエンジンは乙女ゲームジャンルに比類なきリアリズムをもたらし、リアルタイムライティング（Lumen）や極めて詳細なキャラクターモデルを実現しています。ただし、これらの機能、特に4K解像度をターゲットにする場合は、膨大なGPUリソースを必要とします。

Macユーザーの場合、Intel搭載モデルと最新のApple Siliconラインナップではパフォーマンスが大きく異なります。Mシリーズチップは、ユニファイドメモリを搭載しており、UE5が必要とする大量のアセットストリーミングに適しています。「褪せゆく光の街」の高解像度テクスチャを処理するために、少なくとも16GBのRAMを確保することを推奨します。

| コンポーネント | 最小要件 | 推奨（4Kシネマティック） |
| :--- | :--- | :--- |
| **プロセッサ** | Apple M1 または Intel Core i5 | Apple M2 Pro/Max または Intel Core i7 |
| **メモリ** | 8 GB ユニファイドメモリ / RAM | 16 GB 以上 |
| **グラフィックス** | 8コアGPU（内蔵） | 19コアGPU または RTX 3060相当の専用GPU |
| **ストレージ** | 50 GB SSD | 100 GB NVMe SSD |
| **OSバージョン** | macOS Monterey 12.0 | macOS Sequoia 15.0 以降 |
